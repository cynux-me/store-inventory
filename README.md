# Inventory & Accounting Management Software 📊 (Work in Progress)

A custom desktop-based application designed to streamline inventory tracking, sales logging, and financial accounting for small businesses and retail shops (such as hardware stores). Built using Python, Tkinter for the graphical interface, and SQLite for robust local data storage.

> ⚠️ **Note:** This project is currently **under active development**. Features are being added incrementally as part of an ongoing roadmap.

## 🚀 Current Status & Features
* **Database Architecture:** Fully configured SQLite database to handle products, stock levels, and daily transactions.
* **User Interface:** A clean and user-friendly GUI built with Python's Tkinter library.
* **Stock Management:** Basic functionality to add new items, update existing stock levels, and calculate initial profit margins.
* **Search Mechanics:** Implemented custom search suggestions to quickly locate items within the inventory database.
* **Localization:** The user interface (GUI) and application text are fully localized in Bangla, tailored specifically for local business operations.

## 🛠️ Technology Stack
* **Language:** Python 3.x
* **GUI Framework:** Tkinter
* **Database:** SQLite3
